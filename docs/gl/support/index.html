Asistencia e Desenvolvemento¶

Asistencia Comercial¶

Coma desenvolvedor de QField, OPENGIS.ch proporciona paquetes de formación e contratos de asistencia comercial personalizados con SLA

Desenvolvemento e Aplicacións Personalizadas¶

Probablemente teñas un montón de ideas para magníficas características que aínda non foron implantadas. Por favor, ponte en contacto con nós para debater o mellor enfoque para integrar esta idea en QField ou para crear a túa propia app personalizada.

Créditos¶

QField é desenvolvido por OPENGIS.ch. OPENGIS.ch ofrece consultoría, desenvolvemento e formación en software de código aberto incluídos QField e QGIS.

Estabilidade da API¶

QField non se considera actualmente API estable.

Licenza¶

QField entrégase baixo a Licenza Pública GNU (GPL) Versión 2 ou superior. Desenvolver QField baixo esta licenza significa que podes inspeccionar e modificar o código fonte e garante que teñas sempre acceso a unha aplicación de recollida de datos de campo baseada en QGIS que é libre de custos e que se pode modificar libremente.

Podes atopar o código fonte en github
