Datos ráster¶

Os datos ráster poden chegar a ser bastante grandes rapidamente, cando se traballa con ficheiros tiff descomprimidos estes son de varios Gb de datos. En dispositivos móbiles especialmente, isto é ineficiente.

Utilizar Geopackage¶

Recomendamos utilizar o formato geopackage para tratar datos ráster. Os seguintes comandos converterán un ficheiro denominado raster.tif a un ficheiro raster.gpkg con pirámides. Asegúrate de axustar o EPSG:21781 ó SRC desexado.

gdal_translate --config OGR_SQLITE_SYNCHRONOUS OFF -co  APPEND_SUBDATASET=YES -co TILE_FORMAT=WEBP -a_srs EPSG:21781 -of GPKG raster.tif raster.gpkg
gdaladdo --config OGR_SQLITE_SYNCHRONOUS OFF -r AVERAGE raster.gpkg 2 4 8 16 32 64 128 256
