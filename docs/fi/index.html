QField - sinun mobiili [Q]GIS ratkaisusi¶

QField mahdollistaa tehokkaan työskentelyn GIS-datan kanssa kentällä.

QField’s optimized user interface hides the full power of QGIS under the hood.

Yleiset periaatteet¶

QField suunniteltiin muutamaa konseptia silmällä pitäen.

Pidä asiat yksinkertaisena¶

Vaatimukset kentällä eivät ole samat kuin työpöydällä. Näyttö on pienempi, syöttölaitteet ovat erilaisia ja tehtävät ovat erilaisia.

QField aims to help users perform the tasks they need to do without cluttering the user interface. This means, that only tasks which need to be done on the field are availble from the interface. Everything else is not.

Tämä tarkoittaa sitä, että kaikki tasojen tyyleistä lomakemäärityksiin ja muihin projektiasetuksiin tulisi tehdä ensin QGISillä varustetun tietokoneen ääressä.

On yhteensopiva QGISin kanssa¶

QField pohjautuu QGIS-ohjelmistoon. Se ei ole QGISin uudelleen kehitetty versio, vaan se todella käyttää QGISin kirjastoja. Renderöintimoottori on täysin sama kuin QGISissä, joten projektisi näyttää tismalleen samalta niin mobiililaitteellasi kuin tietokoneellasikin.

Jos QGIS-projektissa on saatavilla jo joitain määritysvalintoja, niitä ei tulisi keksiä uudelleen. Tästä syystä QField käyttää samoja muokkaustyökaluja kuin QGIS. Jos projekti on määritelty työasemassa, sen tulisi toimia ongelmitta myös mobiilisti.

Muista, että tämä on vain konsepti. Näitä asioita ajattelemme, kun kehitämme QFieldiä. Tämä ei tarkoita, että kaikki olisi jo täydellisen valmista.

Tilaperusteinen¶

QField is built around modes. Modes are similar to a map tool in QGIS desktop. A mode defines the task which a user is currently doing. Either a user is browsing through the data or he/she is digitizing something new.
