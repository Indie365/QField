Lisää joukko kuvia ominaisuuteen¶

Voit lisätä ominaisuuteen yhden tai useamman kuvan. Alla on esimerkki siitä, miten edetä.

Luodaan kaksi taulua.¶

Tarvitsemme kaksi taulua. Yksi taulu, johon ominaisuudet tallennetaan, ja yksi luettelolle kuvista.

Apiary¶

  • id (UUID)
  • geometry
  • …

Apiary_pictures¶

  • id
  • apiary_id (UUID)
  • path (TEXT)
  • …

Relaatiot¶

Luo relaatio jossa:

  • apiary Viitattu taso
  • id Viitattu tieto
  • apiary_picture Viittaava taso
  • apiary_id Viittaava tieto
  • strength Kooste

Työkalut¶

apiary¶

Aseta kentän tunnuksen oletusarvoksi `` uuid () ``. Sitä ei tarvitse näyttää lomakkeella.

Aseta relaation työkalun arvoksi `` Monet yhdeksi relaatio`` ja lisää relaatio lomakkeelle

kuva¶

Aseta tietopolun widget-tyyppiksi `` liite`` ja lisää se lomakkeelle

Geotagging¶

Some mobile devices will require Open Camera to be installed in order to enable geotagging.

To enable geotagging perform the following steps:

  1. In QField, go to the settings and make sure ”Use native Camera” is activated
  2. Install the Open Camera app on your mobile device
  3. Within the Open Camera settings, make sure geotagging is enabled
  4. Within you mobile device settings, set the default camera app to Open Camera
  5. Completed! Open Camera will now be utilized while taking pictures with QField
