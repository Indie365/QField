Daten aufnehmen¶

Um neue Features zu digitalisieren, aktiviere den Bearbeitungsmodus im Hauptmenü (Stift-Schaltfläche) und wähle den Layer aus, zu dem du neue Features hinzufügen möchtest (die Ebene wird in grün hervorgehoben).

Punkte¶

Navigiere das Fadenkreuz in der Mitte des Bildschirms zu der gewünschten Position und klicke den + Button unten rechts auf dem Bildschirm, um die Erstellung des neuen Punktfeatures zu bestätigen.

Du kannst den Zentriert bleiben Button verwenden, um das Fadenkreuz an deiner Position zu zentrieren, sofern du GPS aktiviert hast.

Linien und Polygone¶

Navigiere das Fadenkreuz in die Mitte des Bildschirms zum gewünschten Start der Linie oder des Polygons und klicke den Plus (+) Button unten rechts auf dem Bildschirm um den ersten Knotenpunkt zu setzen.

Setze weitere Punkte zur Erstellung der Linie oder des Polygons durch klicken des Plus (+) Buttons wenn du einen neuen Knotenpunkt hinzufügen möchtest.

Klicke auf das **Minus (-) Button, um den zuletzt gesetzten Knotenpunkt zu löschen.

Klicke auf den **Schliessen (X) Button um den Objektentwurf zu verwerfen.

Sobald du mindestens 2 Knotenpunkte einer Linie oder 3 Knotenpunkte für ein Polygon hinzugefügt hast, erscheint das **Speichern Button. Klicke auf den Button um den Entwurf zu speichern.

Attributformular¶

After digitizing a geometry, the attribute form will be displayed if it is not suppressed and the user will be asked to enter the attributes for the new feature. The form which appears allows entering attribute values for the new feature.

Remember attribute values¶

For quick collection of rather homogeneus datasets, it is crucial to not having to enter the same attribute values over an over. The checkboxes at the right of every attribute allow remembering each attribute individually so that the next time you will add a feature on the same layer, these attributes will be automatically pre-filled.

Objekte Löschen¶

Das Löschen von Objekten ist nur im Digitalisiermodus möglich.

Um ein Objekt zu löschen, tippe es zuerst an, um es zu identifizieren. In der erscheinenden Identifikationsliste muss du lange auf das entsprechende Objekt drücken, um es auszuwählen. Wenn du möchtest, kannst du nun weitere zu löschende Objekte auswählen. Klicke auf den Papierkorb Button um das Löschen zu bestätigen.

Edit features¶

To edit, tap on Menu button -> pencil.

You can edit an existing feature by moving, adding or deleting the vertex and also spliting.

Split tool¶

Linien- und Polygonobjekte können entlang der gezeichneten Schnittlinie zerschnitten werden.

Vertex tool¶

It is possible to move and delete vertices of geometries and add new ones.

Freehand digitizing¶

The freehand digitizing mode allows users to ‚draw‘ lines and polygons using their stylus/pen or mouse. The mode is available for line/polygon feature additions as well as the geometry editor’s rings tool.

The freehand digitizing mode is activated through a new toolbar button which appears when QField is set to editing mode and a line or polygon vector is selected.

Topological Editing¶

If topological editing is activated on the project QGIS before exporting for QField, shared nodes of neighbouring geometries are modified together when moving / deleting vertices and additional vertices are added to segments when a new node is added from neightbouring geometries.

Multi editing¶

You can make a multi-selection and edit all the features at the same time. In some cases, it can be very efficient.
