Map styling and configuration¶

QField支持大多数QGIS提供的设置项，且无需额外的工作。

样式¶

QField直接支持QGIS中的所有样式设置，包括所有的渲染器类型，如渐进、分类、基于规则、2.5维以及数据定义符号。

显示表达式¶

QField通过名称标识对象。可以通过在QGIS中打开属性表并切换到表单视图来定义生成此名称的表达式。在列表中选择适当的字段或表达式。从QGIS 3开始，可以通过矢量图层属性-->显示页面直接完成此操作。

显示表单式也用于图层内搜索

显示表达式的使用遍及QField之中，用于通过标识字符串表示要素。

只读，不可识别和可搜索图层¶

项目中的一些图层纯粹是为了视觉效果而存在的。当用户点击某个地方以识别对象时，不应该显现这些图层。

一些图层作为信息来源，不应该被用户修改。可以保护这些图层，使之免于被编辑属性或被添加、删除要素。

也可以配置那些图层可以被搜索。

通过项目-->属性-->识别图层页面，激活图层对应的复选框，配置图层的可识别、只读、可搜索等行为。

配置不可识别、不可修改和/或不可搜索图层。

捕捉¶

While digitizing new features, new points can be snapped to existing geometries.

All configuration can be done in Project ‣ Snapping Settings (Settings ‣ Snapping Settings in QGIS 2.x).

捕捉类型¶

可以捕捉新点

  • 仅对已存在几何图形的顶点
  • only to segments of existing geomtries
  • to nodes and segments of existing geomtries

捕捉图层¶

It is also possible to only snap to one or a few layers.

捕捉容限¶

The snapping tolerance can be specified in map units or pixels.

In almost any case, the units should be set to pixels. We made good experiences with a tolerance value of 20.

3D 坐标¶

GPS设备能够测量地球表面2维位置处的高程信息。这些信息可以保存下来。
