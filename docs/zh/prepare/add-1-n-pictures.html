为要素添加多张图片¶

可以为要素增加一张或多张图片。以下为操作示范。

创建两张表¶

需要两张表。一张表存储要素，另一张表存储图片列表。

Apiary¶

  • id (UUID)
  • geometry
  • ...

Apiary_pictures¶

  • ID
  • apiary_id (UUID)
  • path (TEXT)
  • ...

关系¶

创建关系如下：

  • apiary 被参考图层
  • id 被参考字段
  • apiary_picture 参考图层
  • apiary_id 参考字段
  • strength 复合

部件¶

apiary¶

将id字段的缺省值设为 uuid()。该字段不需要在表单中显示。

将关系部件设置为``Many to one relation`` 并将该关系添加到表单中。

图片¶

将path字段的部件类型设为`attachment``并将其添加到表单中。

智能影像定位¶

某些移动设备需要安装 Open Camera 才能启用智能影像定位。

要启用智能影像定位，请执行以下步骤：

  1. 在QField中，转到设置并确保 "使用本机相机" 已激活
  2. 在您的移动设备上安装 Open Camera 应用程序 <https://play.google.com/store/apps/details?id=net.sourceforge.opencamera&hl=en&gl=US>
  3. 在 Open Camera 设置中，确保启用了智能影像定位功能
  4. 在您的移动设备设置中，请将默认相机应用程序设置为 Open Camera
  5. 非常好！现在将在QField中使用 Open Camera 进行拍照
