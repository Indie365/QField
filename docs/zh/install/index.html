安装¶

安卓¶

QField适用于比 Android 5 更高的版本，如果您想购买一个新的设备，我们建议安卓系统版本至少使用 Android 9。

获取预建的apk¶

如果您不能或不想使用GooglePlay，我们还提供了一个已编译apk供您下载。您可以从release页面获得该apk。

Windows¶

Windows支持版本目前处于测试状态。 请下载最新的开发版本 并给我们留言它的运行状况如何，谢谢。

iOS¶

iOS版本正在开发中。您可以在 testflight 中注册Beta测试，以便比其他人更早地获得访问权限。
