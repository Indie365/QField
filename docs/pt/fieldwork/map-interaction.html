Interagir com os mapas¶

Legenda do Mapa¶

Abra o Menu principal e expanda a lista de camadas para exibir a legenda do mapa.

Turning on and off layers¶

Main menu ‣ long press on the layer ‣ check or uncheck show on map canvas

Funcionalidades de identificação¶

Tap on a feature on the map to identify it. If several features are located where you tapped (either because there are multiple features really close one to another, or because several layers are overlapping), they will all be listed in the menu that opens on the right of the screen.

Toque em uma das feições listadas para acessar seus atributos.

  1. Toque nas setas para rolar por todas as feições identificadas.
  2. Toque no botão central para centralizar o mapa na feição selecionada.
  3. Toque no botão de edição para editar os atributos da feição selecionada.
  4. Scroll through the sub-menus to access all the attributes.

Excepções para camadas identificadas¶

Geralmente, não é necessário poder consultar todas as camadas. Algumas camadas estão presentes apenas como mapa base e os seus atributos não são do seu interesse.

Pode gerir a lista de camadas no QGIS Desktop no Projecto ‣ Propriedades do Projecto ‣ Identificar Camadas e desactivar as camadas base.

Measure tool¶

Open the Main menu –> tap the settings button –> Measure Tool
