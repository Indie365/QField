Imprimir em PDF¶

É possível exportar os mapas e outros itens para um documento PDF.

Se o projeto tiver pelo menos um layout de impressão definido, eles estarão disponíveis no menu.

Após selecionar o layout de impressão, o documento PDF é criado e armazenado no diretório de documentos do dispositivo.

Ao escolher a aplicação de leitor de PDF, o documento pode ser aberto diretamente.
