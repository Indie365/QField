Recolher elementos¶

Para começar a digitalizar novos elementos, ative o Modo de edição no menu principal (botão de lápis) e selecione a camada na qual deseja adicionar um novo elemento (a camada será destacada a conzento).

Pontos¶

Navegue pela mira no centro do ecrã até o local desejado e clique no botão + no canto inferior direito da tela para confirmar a criação de um novo elemento de ponto.

Você pode usar o botão manter centralizado para forçar a mira a centralizar na sua localização, se você tiver ativado o GPS.

Linhas e Polígonos¶

Navegue pela mira no centro da tela até o início da linha ou polígono desejado e clique no botão Mais (+) no canto inferior direito do ecrã para adicionar o primeiro nó.

Prossiga com a adição de pontos para formar sua linha ou polígono clicando no botão Mais (+) sempre que desejar adicionar um novo nó.

Clique no botão Menos (-) para remover o último nó adicionado.

Clique no botão Fechar (X) para cancelar completamente a criação do elemento.

Quando adiciona pelo menos 2 nós para uma linha ou 3 nós para um polígono, um botão Salvar será exibido. Clique nele para finalizar seu esboço.

Formulário do atributo¶

Após a digitalização de uma geometria, o formulário do atributo será exibido se não for suprimido e será solicitado ao utilizador a inserção dos atributos para o novo elemento. O formulário que aparece permite introduzir valores de atributo para o novo elemento.

Lembrar os últimos valores¶

Para uma recolha rápida de conjuntos de dados bastante homogéneos, é crucial não ter que inserir os mesmos valores de atributo várias vezes. As caixas de seleção à direita de cada atributo permitem lembrar cada atributo individualmente para que na próxima vez que adicionar um elemento na mesma camada, esses atributos sejam preenchidos automaticamente.

Eliminar elementos¶

Só é possível apagar elementos no modo de digitalização.

Para eliminar um elemento, identifique primeiro clicando com um toque curto. De seguida selecione-o com um toque longo na lista de identificação, selecione mais elementos que querem eliminar e clique no ícone de botão de lixo para confirmar a eliminação.

Editar elementos¶

Para editar, toque no Botão Menu -> lápis.

Pode editar um elemento existente, movendo, adicionando ou eliminando o vértice e também dividindo.

Ferramenta de recorte¶

É possível recortar uma geometria existente através do desenho de uma linha onde pretendemos recortar.

Ferramenta de vértice¶

É possível mover e eliminar vértices das geometrias e adicionar novos.

Digitalização manual¶

A digitalização manual permite aos utilizadores “desenhar” linhas e polígonos usando a sua pen/stylus ou o rato. O modo está disponível para adição de elementos do tipo linha/polígono assim como a ferramenta geometrias de editor de anéis.

O modo de digitalização manual é ativado através de um novo botão que aparece no QField quando é definido no modo edição e a linha ou polígono vetorial é selecionado.

Edição Topológica¶

Se a edição topológica estiver ativada no projeto do QGIS antes de exportar para o Qfield, os nós das geometrias vizinhas partilhadas são modificados simultaneamente quando são movidos /eliminados e os vértices adicionais são adicionados aos segmentos quando um novo nó é adicionado nas geometrias vizinhas.

Multi-edição¶

Pode fazer multi-selção e editar todos os elementos ao mesmo tempo. Em alguns casos, pode ser muito eficiente.
