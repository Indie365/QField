Adicionar um camada fora da pasta do projeto exportado¶

Num projeto QField é possível usar uma camada fora da pasta do projeto, como por exemplo um mapa base.

É útil se um mapa base é usado em todos os projetos, evitando assim copiar várias vezes na pasta do projeto.

No QGIS, abra «Opções» –> «Fontes de dados» –> » Caminhos de Dados Localizados» adicione o caminho á pasta onde está os dados externos.

Para funcionar no dispositivo, adicione as camadas na pasta <device root>/QField/basemaps
