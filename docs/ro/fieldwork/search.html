Căutare¶

Tap on the Search button on the top-right corner of the screen to search for features in the current project.

  1. Tap on the name in the result to pan the map on the feature (the feature will be highlighted in yellow).
  2. Tap on the attributes button to open its attributes.

GoTo Coordinate¶

It is possible to go to coordinates with the search tool.

Tap on the coordinates in the result to go to the coordinates. It is always possible to search WGS84 coordinates (longitude, latitude) or current map CRS (x,y).

Search in value list¶

It is possible to search value in a value list using the magnifying glass next to the field.

To be able to use the „completer” the widget has to be set up Autocomplet form set up.

Căutarea în straturi¶

Puteți căuta entități în straturile vectoriale. În prezent, căutarea se realizează la fel ca și în filtrul de localizare al tuturor straturilor din QGIS (accesibil în bara de localizare QGIS prin utilizarea prefixului af).

Configurarea căutării în QGIS¶

  1. Stabiliți straturile căutabile în Proiect ‣ Proprietățile ProiectuluiP ‣ Identificare Straturi
  2. Definiți expresia de afișare pentru straturi, aceasta fiind utilizată la căutarea entităților. Configurarea ei are loc în pagina Proprietăţile Stratului Vectorial ‣ Afișare
