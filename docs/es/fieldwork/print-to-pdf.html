Imprimir a PDF¶

Es posible exportar los mapas y otros elementos a un documento PDF.

Si el proyecto tiene al menos un diseño de impresión definido, la impresión está disponible en el menú.

Después de seleccionar la composición de impresión, el documento PDF se crear y se guarda en el directorio de documentos del dispositivo.

El documento se puede abrir directamente seleccionando la aplicación de lectura de PDF.
