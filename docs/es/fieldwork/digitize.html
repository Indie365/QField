Recopilar objetos¶

Para comenzar a digitalizar nuevos objetos, active el Modo edición en el menú principal (botón lápiz) y seleccione la capa a la que quiere añadir objetos nuevos (la capa se resaltará en verde).

Puntos¶

Desplace la cruz del centro de la pantalla a la localización deseada y pulse el botón + de la esquina inferior derecha de la pantalla para confirmar la creación del nuevo objeto de punto.

Puede usar el botón Mantener centrado para forzar que la cruz se centre en su localización si tiene activado el GPS.

Líneas y polígonos¶

Desplace la cruz del centro de la pantalla a la localización deseada de inicio de la línea o polígono y pulse el Botón Más (+) de la esquina inferior derecha de la pantalla para añadir el primer nodo.

Continúe añadiendo puntos para formar su línea o polígono pulsando el Botón Más (+) cada vez que quiera añadir un nuevo nodo.

Pulse el Botón Menos (-) para eliminar el último nodo añadido.

Pulse el Botón Cerrar (X) para cancelar por completo la creación del objeto.

Cuando haya añadido al menos 2 nodos para una línea o 3 para un polígono, aparecerá un Botón Guardar. Púlselo para finalizar su diseño.

Formulario de atributos¶

After digitizing a geometry, the attribute form will be displayed if it is not suppressed and the user will be asked to enter the attributes for the new feature. The form which appears allows entering attribute values for the new feature.

Remember attribute values¶

For quick collection of rather homogeneus datasets, it is crucial to not having to enter the same attribute values over an over. The checkboxes at the right of every attribute allow remembering each attribute individually so that the next time you will add a feature on the same layer, these attributes will be automatically pre-filled.

Borrar objetos¶

La eliminación de elementos sólo es posible en el modo de digitalización.

Para borrar un objeto, identifíquelo primero mediante una pulsación corta. Luego selecciónelo con una pulsación larga en la lista de identificación. Seleccione otros objetos a borrar si lo desea y pulse el botón de la papelera para confirmar el borrado.

Editar objetos¶

Para editar, pulse en el botón Menú -> lápiz.

Puede editar un objeto existente moviendo, añadiendo o borrando vértices y también dividirlo.

Herramienta de división¶

Es posible dividir una geometría existente dibujando una línea por donde debe dividirse.

Herramienta de vértices¶

Es posible mover o borrar vértices de una geometría y también añadir otros nuevos.

Freehand digitizing¶

The freehand digitizing mode allows users to “draw” lines and polygons using their stylus/pen or mouse. The mode is available for line/polygon feature additions as well as the geometry editor’s rings tool.

The freehand digitizing mode is activated through a new toolbar button which appears when QField is set to editing mode and a line or polygon vector is selected.

Topological Editing¶

If topological editing is activated on the project QGIS before exporting for QField, shared nodes of neighbouring geometries are modified together when moving / deleting vertices and additional vertices are added to segments when a new node is added from neightbouring geometries.

Multi editing¶

You can make a multi-selection and edit all the features at the same time. In some cases, it can be very efficient.
