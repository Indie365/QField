GNSS¶

Una pulsación corta sobre el botón GNSS encenderá el GNSS y centrará la posición actual una vez que esté disponible la información de posicionamiento.

Activar el modo edición y pulsar el botón diana. La cruz en el centro significa que se está usando posicionamiento GNSS.

Una pulsación larga sobre el botón GNSS mostrará el menú de posicionamiento.

Dentro del menú de posicionamiento puede activar la pantalla de posicionamiento, que mostrará las coordenadas actuales proyectadas en el SRC del proyecto, junto con información de la precisión.

Note

Si ve información en WGS 84 lat/lon en lugar de información en su SRC, probablemente aún no tenga señal.

Using an external GNSS-Receiver¶

QField supports connecting external GNSS antennas via bluetooth.

In settings -> positioning, paired bluetooth devices can be scanned and chosen as position source.

Make sure no other app like mock location providers are using the bluetooth antenna.

Mock location¶

Es posible proporcionar una ubicación simulada de una aplicación android independiente a QField. Hay varias opciones para esto, una de ellas es Android NTRIP Client.

Para usar esto, debe activar las localizaciones simuladas en su dispositivo Android <https://www.youtube.com/watch?v=v1eRHmMiRJQ>`_.
