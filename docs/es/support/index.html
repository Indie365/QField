Soporte y desarrollo¶

Soporte comercial¶

Como desarrollador de QField, OPENGIS.ch <https://www.opengis.ch/#features> _ proporciona paquetes de formación <https://www.opengis.ch/qfield-training/> _ y comerciales personalizados contratos de soporte <https://www.opengis.ch/qgis-support/> _ con SLA

Desarrollo y aplicaciones personalizadas¶

Probablemente tenga muchas ideas para características interesantes que aún no se han implementado. Por favor, contáctenos <https://www.opengis.ch/#contact> _ para discutir el mejor enfoque para integrar esta idea en QField o para crear su propia aplicación personalizada.

Créditos¶

QField es desarrollada por OPENGIS.ch. OPENGIS.ch hace consultoría, desarrollo y formación en software de código abierto, incluyendo QField y QGIS.

Estabilidad de la API¶

QField no considera estable su API actualmente.

Licencia¶

QField se distribuye bajo la Licencia Pública GNU (GPL) versión 2 o superior. El desarrollo de QField bajo esta licencia significa que puede inspeccionar y modificar el código fuente y garantiza que siempre tendrá acceso a una aplicación de recolección de datos de campo basada en QGIS gratuita y que puede ser modificada libremente.

Puedes encontrar el código fuente en github
