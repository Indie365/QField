QField Installation Guide¶

Android Version¶

QField can safely be used with any version later than android 4.3.

Get it on GooglePlay¶

You can download the app directly from GooglePlay.

Get a pre-built apk¶

We also offer a pre-built apk for you to download if you are not able or do not want to use GooglePlay. You can get the qfield.apk from the release page.
