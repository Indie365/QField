GNSS¶

Un court appui sur le bouton GNSS active le GNSS et centre la carte sur la position actuelle une fois l’information disponible.

Activer le mode édition et appuyer sur le bouton en forme de cible, la croix au centre signifie que le GNSS est utilisé pour la position.

Un appui prolongé sur le bouton GNSS ouvre le menu de positionnement.

Dans le menu de géolocalisation, l’affichage des données de position peut être activé. Les coordonnées affichées sont reprojetées dans le SCR du projet.

Note

Si les informations sont affichées en WGS84 au lieu de votre propre SCR, c’est probablement parce qu’aucun signal n’est disponible.

Utilisation d’un récepteur GNSS autonome.¶

QField peut se connecter à des récepteur GNSS autonomes par bluetooth.

In settings -> positioning, paired bluetooth devices can be scanned and chosen as position source.

Make sure no other app like mock location providers are using the bluetooth antenna.

Mock location¶

Il est possible de fournir un emplacement fictif via une application androïd séparée de QField. Il existe plusieurs options pour cela, l’une d’entre elles est le Android NTRIP Client.

Pour l’utiliser vous devez autoriser votre smartphones à utiliser les positions fictives <https://www.youtube.com/watch?v=v1eRHmMiRJQ>`_.
