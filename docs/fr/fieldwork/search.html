Rechercher¶

Appuyez sur le bouton de recherche en haut à droite de l’écran pour rechercher des éléments du projet en cours.

  1. Cliquer sur le nom affiché dans les résultats pour centrer la carte sur l’entité (cette dernière sera par ailleurs colorée en jaune).
  2. Cliquer sur le bouton Attributs pour ouvrir directement les attributs de l’entité.

GoTo Coordinate¶

Il est possible de se rendre aux coordonnées avec l’outil de recherche.

Tapez sur les coordonnées dans le résultat pour vous rendre aux coordonnées. Il est toujours possible de rechercher les coordonnées WGS84 (* longitude, latitude*) ou la carte actuelle CRS (x,y).

Search in value list¶

It is possible to search value in a value list using the magnifying glass next to the field.

To be able to use the « completer » the widget has to be set up Autocomplet form set up.

Rechercher dans les couches¶

Vous pouvez rechercher des entités dans les couches vectorielles. La recherche est identique à « Entités dans toutes les couches » dans QGIS (accessible dans la barre de recherche QGIS en utilisant le préfixe af).

Configurer la recherche dans QGIS¶

  1. Sélectionnez les couches qui seront utilisées pour la recherche dans Projet ‣ Propriétés du projet ‣ Identifier les couches
  2. Définir les « expressions d’affichage » pour les couches concernées, utilisées pour rechercher les entités. À configurer dans la page Propriétés de la couche vecteur ‣ Affichage
