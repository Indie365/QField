Ajouter une série de photos à une entité¶

Vous pouvez ajouter une ou plusieurs photos à une entité. Vous trouverez ci-dessous un exemple sur la manière de procéder.

Création de deux tables¶

Nous aurons besoin de deux tables : l’une où les entités seront stockées et l’autre avec une liste des photos.

Rucher¶

  • id (UUID)
  • geometry
  • …

Apiary_pictures¶

  • id
  • apiary_id (UUID)
  • path (TEXT)
  • …

Relations¶

Créez une relation avec:

  • rucher Couche référencée
  • id Champ référencé
  • apiary_picture Couche de référence
  • Champ de référence apiary_id
  • Composition strength

Widgets¶

rucher¶

Définissez la valeur par défaut de l’ID de champ sur uuid(). Pas besoin de le montrer dans le formulaire.

Réglez le widget de relation sur De un à plusieurs et ajoutez la relation au formulaire

image¶

Définissez sur le widget le type de chemin sur la pièce jointe et ajoutez-le au formulaire

Géolocalisation¶

Certains appareils mobiles nécessiteront l’installation d’Open Camera afin de permettre la géolocalisation.

Pour activer la géolocalisation, suivez les étapes suivantes :

  1. Dans QField, allez dans les paramètres et assurez-vous que « Use native Camera » est activé
  2. Installez l’application Open Camera app sur votre appareil mobile
  3. Dans les paramètres d’Open Camera, assurez-vous que la géolocalisation est activée
  4. Dans les paramètres de votre appareil mobile, réglez l’application par défaut de l’appareil photo sur Open Camera
  5. Terminé ! Open Camera sera désormais utilisé pour prendre des photos avec QField
