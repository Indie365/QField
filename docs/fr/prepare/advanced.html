Itinéraire¶

Cette fonctionnalité permet d’inspecter une liste d’objets qui correspondent à un critère défini. Vous pouvez ensuite naviguer dans ces objets avec les flèches pour vérifier leur statut ou ajuster leurs attributs au besoin.

Pour définir un itinéraire à examiner, vous devez pour l’instant utiliser la console Python. Vous pouvez utiliser n’importe quelle QgsExpression pour définir la liste à examiner. Sélectionnez la couche voulue dans la légende et utilisez le code Python ci-dessous pour ajouter l’itinéraire au projet QGIS.

l=iface.activeLayer()
l.setCustomProperty('qgisMobile/itinerary', 'status <> 1')
