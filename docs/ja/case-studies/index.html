ケーススタディ¶

QFieldプロジェクトでは、QFieldのWebサイトでQFieldのケーススタディ（ユーザーストーリー）を公開することを常に歓迎します。 したがって、特定のプロジェクトや日常生活での問題を解決するためにQFieldを使用した経験を共有していただくよう、関係機関、大学、当局、企業にお願いします。 ケーススタディを共有・公開したい場合は、ご連絡ください。

  • QFieldを使用した生態学的測量の効率性の改善
  • QFieldでの地質のマッピング
  • フィジーのLULCマッピングにおけるQFieldを用いたグラウンドトゥルースデータの収集
  • ワッデン海の鳥類の繁殖状況をマッピングするためにQFieldを活用
  • マラリアを感染させる蚊のグラウンドトゥルースデータ収集作業におけるQfieldの活用
  • QFieldを使用した河川の状態の調査
  • ルワンダの地方給水システムの運営維持管理のためのQGIS/QFieldを用いたデータ収集
  • QFieldを使用したバニラの調査
