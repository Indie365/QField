Open a project¶

To open a project, tap on Menu button –> Option button –> Open Project.

Select the project you want to open by navigating to it, or use the Recent projects and Favorites directories shortcuts.

Обрані теки¶

На першому екрані файлового провідника є частина, що відображає обрані теки. Для того, щоб додати теку до обраних, довгим натиском виберіть бажану теку із списку файлового провідника. Щоб видалити із обраних, довгим натиском виберіть теку із списку обраних.

Advanced project selection¶

QField opens QGIS projects in the «.qgs» or «.qgz» format.

QField використовує налаштований файловий провідник для відкриття проектів. Вибравши «Відкрити проект» із меню QField, файловий провідник відображатиме наступне:

  • Основну теку для збереження файлів.
  • Теки, які залежать від інших додатків, на усіх пристроях збереження даних за умов, якщо додаток може розміщувати у них файли та якщо вони не відображені у основній теці збереження файлів.

Inside these directories, only subdirectories and files with the «.qgs» or «.qgz» extension are shown.

Для QField потрібно, аби файл проекту знаходився на пристрої. Тому, з додатками, які забезпечують доступ до віддалених файлів (наприклад, Google Drive, працювати неможливо. Проте, Ви можете використовувати такі додатки, як syncthing чи nextcloud (чи dropbox та інші), котрі дозволяють синхронізувати віддалений файл на Ваш пристрій.

Зовнішня SD карта¶

Файловий провідник відображає певний вміст картки SD як «лише для читання». Тека додатку відокремлена від решти вмісту картки SD, тому що це єдина тека, до якої додаток може записувати дані. Ця тека:

/Android/data/ch.opengis.qfield/files

Note

Technically, this is due to the fact that Android intents ACTION_OPEN_DOCUMENT and ACTION_OPEN_DOCUMENT_TREE, work and grant read-write access on the content of the returning URI, not on the real linked file. This limitation which makes sense for how Android is structured but currently it is unusable with QField as QField needs to access the real file with a real path.

Увага!¶

Зберігаючи дані у цій теці Ви ризикуєте їх втратити. Тека додатку (/Android/data/ch.opengis.qfield/) буде видалена системою, коли Ви деінсталюєте QField з пристрою. Пам’ятайте про це, якщо зберігаєте Ваші дані на зовнішній картці SD.

Як перетворити зовнішню картку SD у внутрішнє сховище¶

Починаючи з версії Android 6.0 Ви можете використовувати Вашу картку SD як внутрішнє сховище даних пристрою. Це називається Гнучке Зберігання і дозволяє ОС Android форматувати зовнішній накопичувач як постійне внутрішнє сховище. Інформацію на такому зовнішньому носію даних буде зашифровано і його неможливо буде використовувати на іншому пристрої.

Будь ласка, запевніться, що Ви починаєте із порожньою карткою SD. Це необхідно, щоб уникнути будь-якої втрати даних, оскільки в процесі Ваша картка буде відформатована (очищена).

  1. Перейдіть налаштування Вашого телефону й виберіть «Зберігання даних».
  2. Виберіть Вашу картку SD, натисніть на три крапки (у верхньому правому кутку) і виберіть «Налаштування».
  3. Виберіть «Форматувати як внутр.сховище», потім «Очистити та Форматувати».
  4. Ваша картка SD буде зараз відформатована як внутрішнє сховище.
  5. Перезавантажте телефон.

Якщо Ви не перезавантажите телефон, чимало функцій можуть працювати неналежним чином.

Можливо, Гнучке Зберігання недоступне на Вашому пристрої навіть якщо на ньому версія Android 6.0 чи вище. Виробник пристрою міг обмежити функціонал Гнучкого Зберігання. Проте, все ще існують методи обійти обмеження із використанням командного рядку.
