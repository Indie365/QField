Telepítés¶

Android¶

A QField bármelyik Android 5-nél újabb verzióval használható, ha egy új eszközt keres, javasoljuk legalább Android 9-t válasszon.

Használjon egy előkészített .apk fájlt¶

Egy előkészített apk fájl kínálunk letöltésre önnek, ha nem tudja vagy nem akarja a GooglePlay-t használni. A qfield.apk állomány a termék hivatalos weboldalán érhető el.

Windows¶

Windows support is currently in beta status. Download the latest development builds and leave us a note how it works.

iOS¶

Egy iOS verzió készülőben van. Jelentkezhet béta tesztelőnek a testfight-on, hogy hozzáférést kapjon mások előtt.
